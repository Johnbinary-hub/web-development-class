1-Week Website Development intro Class with John Obaloluwa

Learn HTML & CSS by Building Real Projects – with Code with Oba

Day 1 Day 2 Day 3 Day 4 Day 5 Day 6 Day 7

Day 1 – Introduction to Web Development

  • What is a website? (Frontend vs Backend)
  • How browsers work
  • Introduction to HTML & CSS
  • Setting up VS Code and Live Server
  • Basic structure of an HTML document
Mini Project: Create your first web page that displays Hello World!

Day 2 – HTML Basics

  • Headings, Paragraphs, and Line Breaks
  • Links and Images
  • Lists: Ordered and Unordered
  • Tables and Forms
  • HTML attributes (href, src, alt, etc.)
Mini Project: Build a simple personal bio page using headings, paragraphs, and images.

Day 3 – Semantic HTML

  • Meaning of semantic tags
  • Using <header>, <nav>, <main>, <footer>
  • Sectioning content properly
  • Importance for SEO and accessibility
Mini Project: Structure a webpage for a “Tech News Blog” using semantic tags.

Day 4 – CSS Basics

  • How to link CSS to HTML
  • Selectors, properties, and values
  • Colors, borders, and backgrounds
  • Margins and padding (box model)
  • Typography basics (fonts, sizes, spacing)
Mini Project: Style your bio page with colors, borders, and font styles.

Day 5 – CSS Layout & Flexbox

  • Display types (block, inline, inline-block)
  • Introduction to Flexbox
  • Centering items with Flexbox
  • Creating simple navigation bars
  • Responsive layouts with Flexbox
Mini Project: Design a responsive 3-column feature section using Flexbox.

Day 6 – Building a Simple Website

  • Combining HTML and CSS
  • Header, Hero, About, and Contact sections
  • Using divs and containers effectively
  • Adding hover effects and spacing
  • Making the website responsive
Mini Project: Create a small “Portfolio Website” using everything learned so far.

Day 7 – Finishing Touches

  • Improving website design and layout
  • Adding favicon and titles
  • Understanding website publishing (hosting and domain)
  • Showcasing your project
Mini Project: Finalize and present your demo website.
